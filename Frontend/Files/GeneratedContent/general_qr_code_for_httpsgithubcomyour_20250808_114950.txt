**What is a QR Code?**
----------------------

A QR code is a type of matrix barcode that consists of black squares arranged in a square grid on a white background. QR codes can store up to 7,089 characters, making them a convenient way to share information quickly and easily.

**Types of QR Codes**
---------------------

There are several types of QR codes, each with its own unique characteristics and use cases:

### 1. Static QR Code

A static QR code is a fixed code that contains static information, such as a URL, text, or contact details. Once generated, the code cannot be changed.

### 2. Dynamic QR Code

A dynamic QR code is a code that can be edited and updated after it has been generated. This type of code is useful for tracking analytics, managing inventory, or updating information.

### 3. Micro QR Code

A micro QR code is a smaller version of a standard QR code. It can store a limited amount of information and is often used for simple applications like URLs or short messages.
